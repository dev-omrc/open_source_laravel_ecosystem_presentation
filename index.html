Open Source Laravel Ecosystem

¿Qué es Laravel?

Es un framework PHP para artesanos de la web, es Open Source (Licencia MIT) y fue creado en 2011 por Taylor Otwell.
Esta pensado para desarrollar aplicaciones de manera rápida con una sintaxis elegante y expresiva.

¿Qué es un Framework?

Se refiere a una estructura de software compuesta por componentes personalizables e intercambiables para el desarrollo de una aplicación.
En otras palabras, un framework se puede considerar como una aplicación genérica y configurable a la que podemos añadirle las últimas piezas para construir una aplicación concreta.
Por lo general nos liga a una Arquitectura de Software, por ejemplo MVC.​

¿Uso o no uso un Framework?

Al no utilizar un framework basicamente se esta reinventando la rueda cada vez que se desea iniciar con un proyecto web, cosas tan comunes como url amigables, manejo de sesiones, manejo de base de datos pueden tomar mas tiempo del que deberían y al final el tiempo es invaluable, y también se pierde la oportunidad de interactuar con otras personas o comunidades que fácilmente pueden ofrecer soporte a los inconvenientes que se nos puedan presentar.

¿El Framework para Artesanos de la web?

De esta manera se cataloga Laravel, pero para poder comprenderlo primeramente debemos comprender que es un artesano.

¿Qué es un artesano?

Es una persona que hace artesanías, trabajos a mano o con herramientas manuales, entonces se requiere de ciertas habilidades y destrezas para poder realizar este tipo de trabajo.

Laravel es el framework para Artesanos de la web

Laravel tiene como objetivo ser un framework que permita el uso de una sintaxis elegante y expresiva para crear código de forma sencilla y permitiendo multitud de funcionalidades, intenta aprovechar lo mejor de otros frameworks y aprovechar las características de las últimas versiones de PHP (por eso artesanos de la web)

Servidores Web compatibles

  • Apache
  • Nginx

Requerimientos

  • PHP >= 7.1.3.
  • Extensión OpenSSL de PHP.
  • Extensión PDO de PHP.
  • Extensión Mbstring de PHP.
  • Extensión Tokenizer de PHP.
  • Extensión XML de PHP.
  • Extensión Ctype de PHP.
  • Extensión JSON de PHP.

Instalación

Vía composer create-project

composer create-project --prefer-dist laravel/laravel NombreApp

Vía instalador de Laravel

composer global require "laravel/installer"
laravel new NombreApp
